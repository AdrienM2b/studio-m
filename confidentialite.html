Politique de Confidentialité

Dernière mise à jour : 8 juillet 2025

Cette politique de confidentialité explique comment Studio M collecte, utilise, protège et partage vos données personnelles dans le cadre de l’utilisation de ses applications mobiles.

1. Données collectées

Studio M peut collecter certaines données anonymes ou personnelles, selon les fonctionnalités des applications :

  • Données techniques : type d'appareil, système d'exploitation, version de l'app
  • Données d’usage : actions dans l’application, temps d’utilisation
  • Données personnelles (le cas échéant) : adresse e-mail, pseudonyme, préférences utilisateur

2. Finalité du traitement

Les données sont utilisées uniquement pour :

  • Fournir et améliorer les services
  • Corriger les bugs et améliorer la stabilité
  • Mesurer l’audience et les performances
  • Répondre aux demandes de support

3. Consentement

Dans les cas où le consentement est requis (ex. : envoi d'e-mails, stockage de données sensibles), il est toujours demandé de manière explicite au sein de l'application.

4. Partage des données

Studio M ne vend ni ne partage vos données personnelles à des tiers non autorisés. Certaines données peuvent être transmises à des services techniques (hébergement, analytics) dans le strict respect de la confidentialité.

5. Conservation des données

Les données sont conservées uniquement pendant la durée nécessaire aux finalités définies ci-dessus, puis supprimées ou anonymisées.

6. Sécurité

Studio M met en œuvre des mesures de sécurité raisonnables pour protéger vos données contre tout accès non autorisé ou fuite accidentelle.

7. Vos droits

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

  • Droit d’accès, de rectification et d’effacement
  • Droit de limitation ou d’opposition au traitement
  • Droit à la portabilité de vos données

Pour exercer vos droits, contactez-nous à : contact@studiom.com

8. Modifications

Studio M se réserve le droit de modifier cette politique. Toute modification majeure sera notifiée dans les applications concernées ou sur cette page.
